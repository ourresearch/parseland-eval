# parseland-eval

Evaluation harness and dashboard for [parseland-lib](https://github.com/ourresearch/parseland-lib). Scores parseland parsers against a hand-annotated gold standard.

- `eval/` — Python scoring harness (`parseland_eval`), gold standard data, run artifacts
- `dashboard/` — React + Vite dashboard, deployed at [openalex-parseland-dashboard-fe36c419013c.herokuapp.com](https://openalex-parseland-dashboard-fe36c419013c.herokuapp.com/)

See [`eval/README.md`](eval/README.md) and [`dashboard/README.md`](dashboard/README.md) for details.
